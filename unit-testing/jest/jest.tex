\section{ Jest }

\subsection{ A Primer. }
Jest is a test runner created by Facebook to allow for the "Delightful
Javascript Testing".

\subsection{ The Benefits of Jest }
\begin{enumerate}
  \item Fast and sandboxed
  \item Built-in code coverage reports
  \item Zero configuration
\end{enumerate}
